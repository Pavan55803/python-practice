# 🐍 Python Practice Programs

A collection of Python programs covering core programming concepts, written as part of my Python learning journey.

---

## 📁 Files

| File | Topics Covered |
|------|---------------|
| `functions.py` | Functions, *args, filtering, counting |
| `list_operations.py` | Lists, duplicates, type checking, slicing, max value |
| `string_manipulation.py` | Reversing, palindrome, word count, run-length encoding |
| `exception_handling.py` | try/except/finally, ZeroDivisionError, ValueError |
| `regular_expressions.py` | re module, email validation, mobile number validation |
| `copy_concepts.py` | Shallow copy vs Deep copy |
| `misc_programs.py` | Factorial, FizzBuzz, largest of three numbers |

---

## 🛠️ Technologies Used

- Python 3.x
- `re` module (Regular Expressions)
- `copy` module

---

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/yourusername/python-practice.git
```

2. Navigate to the folder:
```bash
cd python-practice
```

3. Run any file:
```bash
python functions.py
python regular_expressions.py
```
